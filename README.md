# Claude Code 求职技能包

两套 Claude Code Skill，覆盖求职全流程。

## 安装

```bash
git clone https://github.com/02Ruichen/resume-website.git
cp -r resume-website/skills-package/job-application-workflow ~/.claude/skills/
```

重启 Claude Code 即可。两个 Skill 互相独立，按需安装。

## 包含的 Skill

- **`job-application-workflow`** — JD 分析 → STAR 重构 → 简历优化 → 面试准备 → SOP
- **`resume-website-builder`** — 对话式修改简历网站 + 构建部署

详细说明见 [使用说明](skills-package/使用说明.md)。
