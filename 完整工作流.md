## 🚀 场景一：在新电脑从零搭建（打开 VS Code 终端粘贴即可）

```bash
# ========== 第1步：克隆项目 ==========
git clone https://github.com/02Ruichen/resume-website.git
cd resume-website

# ========== 第2步：安装依赖 ==========
npm install

# ========== 第3步：启动本地预览 ==========
npx vite --port 3000
# 浏览器打开 http://localhost:3000 即可预览

# ========== 第4步（可选）：测试 API ==========
# 本地 API 需要 Netlify CLI 才能跑，一般用不到
# 直接打开线上管理后台改内容即可
```

## 🛠️ 项目文件结构

```
resume-website/
├── index.html             ← 公开简历页面
├── admin.html             ← 管理登录页

│   ├── resume-data.json   ← 📌 简历数据（改内容就是改这个文件）

│   ├── get-resume.js      ← 读取简历 API

│   ├── update-resume.js   ← AI 修改 API（调用 DeepSeek）
│   └── _data-store.js     ← 数据存储（GitHub API）
├── netlify.toml           ← Netlify 配置
├── vite.config.js         ← Vite 构建配置
└── package.json           ← 项目依赖
```

## 🔑 环境变量（在 Netlify 后台设置，不要改代码）

| 变量名 | 说明 |
|--------|------|
| `ADMIN_PASSWORD` | 管理后台密码 |
| `DEEPSEEK_API_KEY` | DeepSeek API Key |
| `GITHUB_TOKEN` | GitHub 访问令牌（读写仓库用） |

查看/修改：`https://app.netlify.com` → 找 resume-website → Site configuration → Environment variables

## ❗ 常见问题

| 问题 | 解决 |
|------|------|
| 管理后台打不开 | 确认网址是 `/admin` 不是 `/admin.html` |
| AI 修改没生效 | Ctrl+F5 强制刷新公开页面 |
| git push 失败 | 检查网络，或等几分钟重试 |
| 忘记密码 | 去 Netlify 环境变量里看 `ADMIN_PASSWORD` |
