# Claude Code 求职技能包 · 使用说明

两套 Skill 覆盖从 JD 分析到简历投递的求职全流程。

---

## 包含哪些 Skill

| Skill | 用途 | 怎么触发 |
|-------|------|----------|
| `job-application-workflow` | JD 分析 → 简历优化 → 面试准备 → SOP | "帮我分析这个 JD""准备面试" |
| `resume-website-builder` | 对话式修改简历网站 + 部署 | "改简历""更新技能栏""重新部署" |

---

## 安装

```bash
# 1. 下载
git clone https://github.com/02Ruichen/resume-website.git

# 2. 复制 skill（挑你需要的装）
cp -r resume-website/skills-package/job-application-workflow ~/.claude/skills/

# 3. 重启 Claude Code
```

> 两个 Skill 互相独立，装一个或两个都行。

## `job-application-workflow` 详解

### 能做什么

六个 Phase 按顺序执行：

| Phase | 做什么 | 产出 |
|-------|--------|------|
| 1 | JD 高频能力拆解 | 硬技能 / 软技能 / 核心诉求清单 |
| 2 | 项目经历 STAR 重构 | 2-4 段植入关键词的量化项目 |
| 3 | 四维简历优化 | 优化版简历（个人简介+技能栏+项目+关键词） |
| 4 | 面试准备 | 1min30s 自我介绍 + ≥12 道结构化面试题 |
| 5 | 可复用 SOP | 标准化投递 Checklist |
| 6 | 同步到简历网站 | 更新数据 → 构建部署（需配合 builder） |

### 需要准备什么

- 一份 **JD**（文字 / 截图 / 链接都行）
- 一份**简历**（PDF / 文字都行）

### 怎么用

直接在 Claude Code 里说：
- 「帮我分析这个 JD，然后准备面试」
- 「秋招投 XX 公司，走一遍全流程」

Skill 会自动激活，缺什么材料会主动问你。

## `resume-website-builder` 详解

### 能做什么

对话式操作简历网站：改文字 → 自动验证排版 → 构建 → 部署上线。

### 需要准备什么

- 一个用 JSON 驱动的简历网站项目（参考本仓库结构）
- Python 环境

### 怎么用

- 「帮我把第二段实习的描述改一下」
- 「技能栏更新，加一个数据治理」
- 「重新构建部署一下」

> 本 Skill 默认对接 `resume-data.json` + `build.py` 的架构，如果你用的是其他简历系统，改一下 SKILL.md 里的项目路径和字段映射就行。

## 两个 Skill 怎么配合

```
/job-application-workflow
    → 分析 JD → 优化简历内容

/resume-website-builder  
    → 把优化内容写进网站 → 构建部署 → 导出 PDF 投递
```

## 常见问题

**Q: 我只想用求职流程，不需要简历网站，可以吗？**

可以，只装 `job-application-workflow`，Phase 1-5 完全独立运行。

**Q: 我的简历不是 JSON 驱动的，能用 builder 吗？**

改一下 SKILL.md 里的构建命令和文件路径就行，核心逻辑不变。

**Q: Windows 上能用吗？**

能。`redskill` CLI 在 Windows 上不可用，直接手动复制 Skill 文件夹即可。
